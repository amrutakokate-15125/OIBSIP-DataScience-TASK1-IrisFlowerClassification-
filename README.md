# Iris Flower Classification

## Objective

Develop a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica species.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Models

* Logistic Regression
* Random Forest Classifier

## Results

Random Forest achieved the highest accuracy and was selected as the final model.

## Key Findings

* No missing values in the dataset.
* Petal Length and Petal Width were the most important features.
* The model successfully classified all three Iris species.
